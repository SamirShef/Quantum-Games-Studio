Что такое Coddy?

Coddy Lang - новый язык программирования от компании Quantum Games Studio. Coddy совмещает в себе элементы синтаксиса из языков C#, Java, C++ и других.

Как пользоваться Coddy?

Чтобы пользоваться языком программирования Coddy необходимо:

  • Зайти на репозиторий проекта и ознакомиться с инструкцией (базовый синтаксис и поддержка)
  • Установить платформу .NET 8.0 для корректной работы компилятора
  • Установить компилятор языка Coddy из репозитория проекта

Примеры кода

Вывод "Hello world!" в консоль

println("Hello world!");

Импорт библиотек

include "math.cd"; println(Math.sqrt(25.0f)); // Выведет 5
